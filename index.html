Distributed by design

Two machines.
One topology.

The headline differentiator. A single lab spans several cheap PCs — Netplex stitches their virtual switches together with VXLAN tunnels, so a link between nodes on different machines just works. Big labs no longer need a big server.

  • Scheduler places nodes by free CPU / RAM across hosts
  • A host dropping degrades only its nodes — the lab keeps running
  • Each host gets a colour; nodes glow to show where they live

The link doesn't care which machine a node lives on.

The image pipeline

Drop an image. Walk away.

EVE-NG made you SSH in, juggle files and run fixpermissions. Netplex does the whole thing — convert, set permissions, organise, boot-test — and shows you every step. No CLI.

Auto-classify

Matches the upload against 138 device kinds across 67 vendors and gives it a human name — never vios-adventerprisek9-m.SPA…

Disk-guarded

Refuses an upload before it fills your disk: "Only 8.2 GB free — needs ≥10 GB." A precise, actionable message, never a stack trace.

Boot-tested

Every image boots headless before it's marked Ready. If it fails, you get what happened, why, and what to do — plus "email me the fix."
